Tienda web de compra venta de artículos de segunda mano con diseño responsive,
implementada con html, js y css.
Se utilizan peticiones AJAX y API Fetch a un servidor RESTful proporcionado
para la carga de los recursos, evitando así la recarga del sitioweb para 
actualizar los contenidos.
Se ha hecho uso del servidor gratuito XAMPP, que mediante la herramienta
phpMyAdmin nos ha permitido importar un script sql para la creación y gestión
de la base de datos de nuestra tienda web.
